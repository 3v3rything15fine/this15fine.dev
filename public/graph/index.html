About This Graph

This is a network analysis of an Obsidian knowledge vault maintained by a doctoral student in Curriculum & Instruction at Texas State University. The vault uses a flat, inquiry-driven graph structure — thirteen inquiry notes serve as organizing questions, each linking to literature, concepts, methods, and projects.

The graph is generated by a local Python pipeline (NLTK + NetworkX) that extracts wikilinks from the vault, detects communities using modularity optimization, computes betweenness centrality, and identifies structural gaps between clusters. No data leaves the local machine.
